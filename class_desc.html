Intro To Programming | CS 149

Easy | Python | 4 hours/week

Pretty easy class that requires minimal studying. Coding projects are decently hard and will take up the majority of the class.

Advanced Programming | CS159

Medium | Java | 6 hours/week

Professor: Wang

The midterms and final are the hardest part of the class. Projects are time-consuming but not as difficult as future classes.

Algorithms and Data Structure | CS240

Hard | Java | 7 hours/week

Professor: Wang

Tests were pretty difficult just make sure to do the labs and study. 4 projects that were very time-consuming and difficult.

Hard | Java | 10 hours/week
